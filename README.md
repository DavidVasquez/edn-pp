# edn-pp

EDN command-line pretty printer

## Installation

```make install```

## Usage

curl http://some-server.com/employees | edn-pp

```
{:data
 [{:first_name "Jim",
   :last_name "Halpert",
   :occupation "Sales Representative"}
  {:first_name "Pam",
   :last_name "Beesly-Halpert",
   occupation "Reception"}
  {:first_name "Michael",
   :last_name "Scott",
   :occupation "Regional Manager"}
  {:first_name "Dwight",
   :last_name "Schrute",
   :occupation "Sales Representative"}]}
```
